插件
-------

`Advanced-Alchemy <https://docs.advanced-alchemy.litestar.dev/latest/>`_ 库提供了一系列插件来添加对 SQLAlchemy 的支持：

- :class:`~advanced_alchemy.extensions.litestar.SQLAlchemyPlugin`：完整的 SQLAlchemy 支持
- :class:`~advanced_alchemy.extensions.litestar.SQLAlchemyInitPlugin`：应用程序工具
- :class:`~advanced_alchemy.extensions.litestar.SQLAlchemySerializationPlugin`：序列化支持
