开发基本 TODO 应用程序
===================================

.. admonition:: 本教程适合谁?
    :class: info

    本教程旨在让你熟悉 Litestar 的基本概念。
    如果你之前没有 Litestar 或一般 Web 框架的经验,
    这是正确的起点。

.. note::
    需要 Python 和 Web 开发概念的基础知识。


本教程的目标是开发一个具有以下功能的 TODO 应用程序:

1. 检索 TODO 列表
2. 向 TODO 列表添加项目
3. 将 TODO 项目标记为*已完成*

这不是很多功能,但足以学习 Litestar 的大多数基本构建块。
