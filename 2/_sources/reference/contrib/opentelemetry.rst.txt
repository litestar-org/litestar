opentelemetry
=============

.. deprecated:: 2.22.0
    The OpenTelemetry integration has moved to :mod:`litestar.plugins.opentelemetry`. The
    ``litestar.contrib.opentelemetry`` namespace remains as a deprecation shim and is
    removed before 3.0.0. See :doc:`/reference/plugins/opentelemetry` for the
    current API documentation.
